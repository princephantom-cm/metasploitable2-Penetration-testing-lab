# Screenshots

This folder contains screenshots demonstrating each step of the penetration testing process performed on the Metasploitable2 lab environment.

The screenshots include:

* Lab environment setup
* Network connectivity verification
* Nmap service enumeration
* Metasploit exploitation
* Root access confirmation
* Web application testing (DVWA)
* SQL Injection exploitation

Each screenshot is numbered to represent the sequence of the penetration testing workflow.
