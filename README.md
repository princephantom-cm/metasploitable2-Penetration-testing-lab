# Metasploitable2 Penetration Testing Lab

This project demonstrates a basic penetration testing workflow performed on the Metasploitable2 vulnerable machine using Kali Linux.

## Lab Environment

Attacker Machine: Kali Linux  
Target Machine: Metasploitable2  
Network Mode: Host-only  

## Tools Used

- Nmap
- Metasploit Framework
- DVWA

## Steps Performed

1. Lab setup using Kali Linux and Metasploitable2
2. Network connectivity verification
3. Service enumeration using Nmap
4. Exploitation using Metasploit
5. Web application testing on DVWA

## Vulnerabilities Exploited

- vsftpd 2.3.4 Backdoor
- Samba Username Map Script
- SQL Injection (DVWA)
